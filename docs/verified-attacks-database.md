# Verified Cyber Attacks Database

This file contains the verified dataset of European cyber incidents used in ORYX. Each entry has been cross-checked against institutional sources (ENISA, CERT-EU, CISA, NCSC) and reputable private sector reports.

> **Status**: Work in progress. Entries marked with [VERIFIED] have been fully confirmed. Entries marked with [PENDING] need additional source confirmation.

---

## Entry Template

For each attack, the following fields must be verified:

- **Name**: Operation or campaign name
- **Date**: Start date (and end date if applicable)
- **Target country**: Primary country/countries affected
- **Target sector**: Sector classification (ENISA taxonomy)
- **Attack type**: Ransomware / Espionage / DDoS / Wiper / Supply Chain / Hack-and-Leak
- **Attributed actor**: Threat actor or APT group (if attributed)
- **Confidence**: Confirmed / Probable / Suspected / Unknown
- **Severity**: Low / Medium / High / Critical
- **Estimated cost**: Economic impact in EUR/USD (with source)
- **Description**: 2-3 sentence summary
- **Primary sources**: Institutional reports with URLs
- **Secondary sources**: Private sector analyses with URLs
